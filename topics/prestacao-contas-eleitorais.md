# Prestação de contas eleitorais

Campaign finance accounting for candidates (municipal elections in the
original note). For **municipal fiscal accounts** (mayoral contas de
governo / gestão), see `contas-municipais.md`.


  - Needs to be done by about Nov 1st (for 1st round)
  - Each candidate needs to have a CNPJ
  - Open bank account for campaign
  - All income needs to be noted in Sistema de Prestação de Contas
    Eleitorais (SPCE) within [72](lei/7.org::*Art%202) hours,
    accompanied by recibo eleitoral
  - Receipt unnecessary if less then R$4,000
  - All expenditure has to be done by cheque or bank account (except if
    less than R$300)
  - Limits on how much you can spend
      - In [2016](lei/201.org::*Art%206) elections, max
        [70](lei/7.org::*Art%200)% of max expenditure in
        [2012](lei/201.org::*Art%202) for same position in same munic
  - Ruled by electoral courts
      - aprovação (com ressalvas), desaprovação, não prestação
  - If não prestação: lose position, if party lose party fund

[source](http://www.politize.com.br/prestacao-de-contas-eleitorais/)
