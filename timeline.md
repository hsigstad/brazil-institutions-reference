# Timeline — institutional changes affecting Brazilian research

A unified, date-anchored timeline of statutory and jurisprudential
changes that matter for empirical research on Brazilian institutions.
Each entry: date, event, brief description, affected file(s) where the
substance is discussed.

This is a **lookup table**, not a textbook. For the substance, follow
the file links.

**Topics / keywords**: timeline, dates, reforms, statutory cutoffs,
institutional changes, structural breaks, regime shifts.

**Snapshot as of 2026**: covers from 1988 (CF) through 2024 (CNJ Res.
547). New events should be added in chronological order.

---

## Coverage of changes by file

If you're working with data spanning a particular period, the files
most likely to be affected by reforms in that period:

| Period | Most affected files |
|---|---|
| 1988 | All (constitutional foundation) |
| 1990s | improbidade, processo-civil, processo-eleitoral, anticorrupcao-penal |
| 2000–2010 | federalismo-fiscal (LRF), contas-municipais (Ficha Limpa), licitacoes (pregão) |
| 2010–2017 | All electoral files, anticorrupcao-penal (Lava Jato era begins) |
| 2017–2021 | partidos-e-sistema-eleitoral (EC 97), justica-trabalho (reforma 13.467 + ADI 5766), improbidade (Lei 14.230 + Tema 1199) |
| 2021–2026 | licitacoes (Lei 14.133 transition), federalismo-fiscal (EC 132), cnj-administracao-judicial (Res. 547) |

| Date | Event | Effect | Files |
|---|---|---|---|
| 1988-10-05 | **Promulgação da Constituição Federal** | New constitutional framework. CF Art. 14 §§9–11 (inelegibilidade, AIME). CF Arts. 70–75 (controle externo). CF Arts. 127–130 (Ministério Público). CF Art. 31 §2 (2/3 rule on contas municipais). | All |

## Key cutoffs as design tools

The following dates frequently anchor empirical research designs:

- **2003-01-01**: CGU random audit program begins → quasi-random treatment.
- **2010-06-04**: Ficha Limpa → expanded ineligibility pool.
- **2013-11-28**: EC 76/2013 → federal precedent for voto aberto reforms.
- **2015-09**: STF ADI 4650 → corporate donation ban (effective 2016).
- **2015-12-31**: CGU random audit program ends → end of clean RD design.
- **2016-08-10**: STF Tema 157 decided → câmara becomes the legally salient decision point.
- **2017-11-11**: Reforma trabalhista effective → sharp filing-cost shock.
- **2018-08**: First election under cláusula de desempenho.
- **2019-11**: ADC 43/44/54 → release of high-profile defendants.
- **2020 municipals**: First election under EC 97/2017 (no proportional coligações).
- **2021-10-20**: STF ADI 5766 → partial reversal of 2017 labor reform.
- **2021-10-26**: Lei 14.230/2021 → improbidade reform.
- **2022-08-18**: STF Tema 1199 → retroactivity rule for the LIA reform.
- **2023-12-30**: Lei 14.133/2021 mandatory → end of Lei 8.666 era.
- **2024**: CNJ Res. 547/2024 → mass extinction of execuções fiscais.

For details on any of these as quasi-experiments, see
[`quasi-experimentos.md`](quasi-experimentos.md).
