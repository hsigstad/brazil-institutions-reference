# Siglas — acronym registry

Alphabetical registry of Brazilian institutional acronyms used
throughout this directory. Each entry: full name, one-line function,
file pointer where the acronym is discussed in depth (when applicable).

This file is built for fast grep: search any acronym to get
disambiguation and a pointer to the substantive file.

**Topics / keywords**: acronyms, siglas, abbreviations, institutional
names, glossary.
